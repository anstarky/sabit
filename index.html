Architecture Workshop

Boost your design skills with a professional Rhino Modeling Courses that will help you to see another vector of Architectural Design.

Course Description

Class 1. Introduction to NURBS modeling.

  • Program Interface
  • Setting up program windows and working field
  • Rhinoceros work environment
  • Setting the display modes
  • Creation and editing of simple objects

Class 2. Curves and Drafting.

  • Introduction to Curves
  • Axes and snaps
  • Workflow with curves
  • Functional component of lines and geometry
  • Architectural Drafting in Rhino
  • Final task: Provide 3 drawings:plan, elevation, section.

Class 3. Drawing design.

  • Worksheet creation
  • Size grid and drawing axes
  • Text and footnote creation
  • Line types and line settings
  • Printing
  • Final task: Provide formal drawings

Class 4: Creating and editing surfaces.

  • Surface creation
  • Geometry analysis
  • Surface treatment
  • Surface application
  • Surface Panelling
  • Final task: Provide a small pavilion model

Class 5. Complex geometry and transition to solid objects.

  • Types of objects and their classification
  • Solid objects
  • Surface integration
  • Functional components of solids and working with them
  • Mesh modeling
  • Final task:To develop a conceptual model of a multi-storey building

Class 6. Developing and detailing a conceptual idea.

  • Rhino's methodology for developing ideas
  • Working with Unishapes
  • Point modeling
  • Sheath creation
  • Details
  • Final task: Provide development of idea with details

Class 7. Terrain and environment.

  • Creation of Terrain and Environment
  • Lands Design
  • Fixing the project to the ground
  • Final task: Binding the project to the terrain and creating an environment

Class 8. Visualization and adjustment of light.

  • Materials and skins at Rhino
  • Light setting
  • Visualization and adjustment of digital production
  • Introduction to Grasshopper
